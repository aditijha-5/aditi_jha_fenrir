DRIFTWOOD ARCADE BROWSER GAME PORTAL
Category: Web Development

SCOPE OF WORK

The work covers the complete front end build of a responsive, multi page gaming
website using HTML, CSS, and plain JavaScript. The site is a self contained
static project with no backend, no database, and no paid plugin.

IN SCOPE

1. Six linked pages that share one header, one footer, and one visual theme:
   home, games catalogue, high score board, about, contact, and one game page
   per playable game.

2. Three original mini games, each rendered on an HTML5 canvas:
   - Dune Dash, a one button endless jumper that dodges obstacles.
   - Star Catcher, a basket game that catches falling stars and avoids bombs.
   - Tide Match, a four by four memory game that matches pairs of icons.

3. A games catalogue page that reads the supplied JSON dataset and shows one
   card per game, with a category filter and a search by title.

4. A high score board page that reads each game best score from local storage
   and shows the values in one table.

5. A contact page with a styled form that validates its fields in the browser
   and confirms on submit without sending data anywhere.

6. A responsive layout that works from a 320 pixel wide phone screen up to a
   wide desktop, with a collapsing navigation menu on small screens.

7. Use of the supplied copyright free sprite and icon assets from the assets
   folder, plus the supplied games dataset.

8. A README file that explains how to run the site on a local static server.

OUT OF SCOPE

1. Any backend, server, database, or login system.
2. Online or multiplayer features, real time chat, or shared leaderboards.
3. Payment, advertising, or in game purchase features.
4. Paid plugins, paid themes, or paid fonts.
5. Frameworks or build tools such as React, Vue, Angular, jQuery, or bundlers.
6. New game art beyond simple recolouring of the supplied copyright free assets.

DELIVERY

The complete source folder of HTML, CSS, JavaScript, the dataset, the assets,
and the README, delivered as a single archive that opens and runs when served
from a static file server.
